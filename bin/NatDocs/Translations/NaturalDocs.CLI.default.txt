CommandLine.SyntaxReference(version).multiline {{{
Natural Docs, version {0}
http://www.naturaldocs.org
This program is licensed under the AGPL
---------------------------------------

Syntax:

    NaturalDocs [project config folder]

    NaturalDocs -i [input (source) folder]
                -p [project config folder]
                -o [output format] [output folder]
                [additional options]

Examples:

    NaturalDocs C:\My Project\Natural Docs Config
    
    NaturalDocs -i /project/src
                -o html /project/doc
                -p /project/ndconf


Parameters:
--------------------

 -p [folder]
--project-config [folder]
    Specifies a folder to store project configuration files.  Required.
    If you run Natural Docs and only specify an empty project configuration
    folder on the command line, it will create default configuration files for
    you.

 -i [folder]
--input [folder]
--source [folder]
    Specifies an input (source) folder.  Required either on the command line
    or in Project.txt.  Can be specified multiple times.

 -o [format] [folder]
--output [format] [folder]
    Specifies an output format and folder.  Required either on the command
    line or in Project.txt.  Can be specified multiple times.

 -xi [folder]
--exclude-input [folder]
--exclude-source [folder]
    Excludes an input (source) folder.

 -xip [pattern]
--exclude-input-pattern [pattern]
--exclude-source-pattern [pattern]
    Excludes input (source) folders by a pattern, which can include * and ?
    wildcards.  The pattern must match an entire folder name, so "cli" will
    not match "client", but "cli*" would.

 -img [folder]
--images [folder]
    Specifies an image folder.

 -t [width]
--tab-width [width]
    Specifies how many spaces a tab should be expanded to.

 -do
--documented-only
    Only include documented code elements in the output.

 -nag
--no-auto-group
    Turns off automatic grouping.

 -w [folder]
--working-data [folder]
    Specifies a folder to store temporary data.  If not specified it will be
    a subfolder of the project configuration folder.

 -r
--rebuild
    Rescans every source file and rebuilds all the output.

 -q
--quiet
    Suppresses all non-error output.

 -v
--version
    Displays the current version of Natural Docs.

 -?
 -h
--help
    Displays this command line reference.
}}}
